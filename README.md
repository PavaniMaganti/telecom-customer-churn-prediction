# 📊 Telecom Customer Churn Prediction & Retention Analysis

## 🚀 Project Overview

Customer churn is a critical challenge for telecom companies as losing customers directly impacts revenue and profitability.

This project uses **Exploratory Data Analysis (EDA)** and **Machine Learning** to identify customers at risk of churn and uncover the key factors influencing customer retention.

---

## 🎯 Project Objectives

✅ Analyze customer churn patterns

✅ Identify key churn drivers

✅ Build predictive machine learning models

✅ Compare model performance

✅ Generate actionable business recommendations

---

## 📂 Dataset Overview

* 👥 7,043 Customer Records
* 📊 50 Original Features
* 📉 Customer Churn Information
* 🌐 Internet Service Details
* 💳 Billing & Payment Data
* ⭐ Customer Satisfaction Metrics

---

## 🧹 Data Preparation

The dataset was prepared using the following steps:

* Handled missing values
* Removed data leakage features
* Removed high-cardinality features
* Encoded categorical variables
* Performed feature engineering
* Created train-test datasets for modelling

---

## 🔍 Key Business Insights

📌 26.54% of customers churned

📌 Month-to-month contracts showed the highest churn rates

📌 Lower customer satisfaction significantly increased churn risk

📌 Customers with shorter tenure were more likely to leave

📌 Higher monthly charges were associated with increased churn

📌 Fiber Optic customers exhibited higher churn rates than other internet service groups

---

## 🤖 Machine Learning Models

Three classification models were evaluated:

### Logistic Regression

* Accuracy: **96.38%**

### Random Forest

* Accuracy: **95.32%**

### XGBoost

* Accuracy: **96.24%**

🏆 **Best Performing Model: Logistic Regression**

---

## 📈 Key Churn Drivers

The most influential factors affecting customer churn were:

🥇 Customer Satisfaction Score

🥈 Tenure in Months

🥉 Total Revenue

🏅 Number of Referrals

🏅 Monthly Charges

🏅 Contract Type

🏅 Internet Type (Fiber Optic)
